# Kraby Game Engine

A lightweight 2D action RPG engine built with TypeScript and PixiJS.

## Stack

- TypeScript
- PixiJS v8
- Vite
- Tauri (for desktop builds)

## Features

- Entity-based architecture (probably ECS in the future)
- Command pattern input system
- Collision and attack colliders
- Animation controller
- Modular gameplay components

## Running the project

To install the dependencies

> npm install

To run the web version

> npm run dev

To run the desktop version (this also runs the web version simultaneously)

> npm run tauri dev

## Notes

Game assets were removed from the public repository.
Create your own Assets directory and load your atlases.
